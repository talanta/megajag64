MegaJag64
===

Getting started :  
```sh
## initil setup (after cloning the repo)
npm install

## run the watcher.. (make sure you have typescript installed globally : npm i -g typescript)
tsc -w

## serving the files
./node_modules/.bin/lite-server --baseDir="./"

## the open in the browser : http://localhost:3000/src/index.html 
```
